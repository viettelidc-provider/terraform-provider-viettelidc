## Schema

### Required

- `instance_id` (String) Database instance ID to bind the backup schedule to.
- `location` (String) Backup location (block, object).
- `max_record` (Number) Maximum number of backup records to keep (1 to 35).
- `name` (String) Scheduler name.
- `scheduler_type` (String) Scheduler type/cycle (e.g. daily, weekly, monthly).
- `time` (String) Start time formatted as 'YYYY-MM-DD HH:mm:ss' (in local Vietnam ICT time).

### Optional

- `description` (String) Scheduler description.
- `is_delete_record` (Boolean) Set to true to delete backup records when the scheduler is deleted.
- `vpc_id` (Number) VPC ID. Uses provider default if not specified.

### Read-Only

- `id` (Number) Backup scheduler ID assigned by the system.
- `status` (String) Current status of the scheduler (e.g. ACTIVE, INACTIVE).
