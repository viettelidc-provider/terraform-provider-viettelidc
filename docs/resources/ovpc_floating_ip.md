# viettelidc_ovpc_floating_ip (Resource)

Allocates a ViettelIDC Floating IP and associates it with a VM instance and NIC. Destroying the resource disassociates and releases the IP.

## Example Usage

```terraform
resource "viettelidc_ovpc_floating_ip" "fip" {
  instance_id          = viettelidc_ovpc_instance.vm.id
  network_interface_id = viettelidc_ovpc_instance.vm.root_nic_id
  vpc_id               = viettelidc_ovpc_vpc.main.id
}
```

## Schema

### Optional

- `id` (String) Floating IP ID (vttFloatingId). When set, the resource associates this existing Floating IP instead of allocating a new one.
- `instance_id` (String) ID of the VM instance to associate with. Changing this value forces a new resource.
- `network_interface_id` (String) ID of the NIC to associate with. Changing this value forces a new resource.
- `vpc_id` (String) VPC ID. Falls back to the provider default vpc_id when unset.

### Read-Only

- `public_ip` (String) Public IPv4 address allocated by the system.
