# viettelidc_ovpc_launch_template (Resource)

ViettelIDC Launch Template for Autoscale Groups. All attributes are immutable (RequiresReplace / ForceNew) — any change destroys the template and creates a new one.

## Example Usage

```terraform
resource "viettelidc_ovpc_launch_template" "web" {
  name        = "web-template"
  description = "Template for web servers"
  vm_id       = viettelidc_ovpc_instance.base_vm.id
  memory_size = 4
  cpu_size    = 2
  vpc_id      = viettelidc_ovpc_vpc.main.id
}
```
