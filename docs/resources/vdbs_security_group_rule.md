# viettelidc_vdbs_security_group_rule (Resource)

Provides a VDBS Security Group Rule resource to manage inbound rules for Database Service.

> **Note:** VDBS currently only supports managing inbound rules.

## Example Usage

```terraform
terraform {
  required_providers {
    viettelidc = {
      source = "viettelidc/viettelidc"
    }
  }
}

resource "viettelidc_vdbs_security_group_rule" "allow_mysql" {
  security_group_id = "123"

  name          = "allow-mysql-inbound"
  type          = "MySQL"
  protocol_name = "TCP"
  port          = "3306"
  source        = "custom"
  source_ip     = "10.0.0.0/16"
}
```
